LingQ to Memrise

LingQ is a great website for compiling lists of words that you need to learn. Memrise is an another website that excels in helping you memorise lists of words. Wouldn't it be great if you could have the best of both worlds? Unfortunately Memrise doesn't provide an API for importing lists of words automatically but it does provide a section where you can paste in a spreadsheet to add words to lessons in bulk. This website aims to help out a little bit by using the amazing LingQ API to format lists of LingQs in a way that you can paste them into Memrise and learn them.
